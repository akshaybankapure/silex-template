# Silex Template

Simple Silex template

### Features
- Silex 2
- Multilingual
- YAML configs
    - Environments support
    - Routes
    - locales
    - Sent to twig as `config`
- HTTP cache
    - Clear route
- Ajax

### Installation
- Install composer
- use `composer install` on root folder
- update `public/.htaccess` RewriteBase line if needed
